## 📝 Smart Formatting Syntax

The "secret sauce" of ExamAce is how it parses questions. Use these markers in your JSON data to trigger crazy formatting:

| Marker | Effect | Example |
| :--- | :--- | :--- |
| `**text**` | **Must-Know Highlight** | `**PEAS**` becomes a glowing badge. |
| `\n` | **Paragraph Break** | Separates dense text into digestible blocks. |
| `- ` | **Smart Bullet** | Creates an animated list item with custom icons. |
| `Example:` | **Call-out Box** | Renders the text in a specialized neon border box. |
| `Definition:` | **Formal Badge** | Highlights formal academic definitions. |
